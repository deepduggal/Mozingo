Mozingo

Modern default colors for CSS Preprocessors.

Installation (Coming Soon)

NPM

npm install mozingo

Bower

bower install mozingo

Basic Usage

Specify the color you want to use as an SCSS variable. In this case, we are using the color red.

.someDiv {
        height: 50px;
        width: 50px;
        background: $red;
      }

And you'll get the better looking version of it. That's it.

When creating a webpage, web developers are often in need of a darker or lighter version of the same color.

Advanced Usage

Mozingo makes it easy to darken or lighten any color.

Dark Colors

.someDiv {
        height: 50px;
        width: 50px;
        background: $redDark;
      }

Light Colors

.someDiv {
        height: 50px;
        width: 50px;
        background: $redLight;
      }
